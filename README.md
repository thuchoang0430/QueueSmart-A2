# QueueSmart A2 Front-End

Improved UI/UX version of the QueueSmart front-end demo.

## Tech Stack
- React
- TypeScript
- Vite
- HTML/CSS

## Demo Accounts
- User: `user@test.com` / `password`
- Admin: `admin@test.com` / `password`

## How to Run
```bash
cd frontend
npm install
npm run dev
```

## Build Check
```bash
npm run build
```

## Notes
This is a front-end-only A2 demo. Queue data, service data, login, and registration use mock/static state because the backend is not required yet.
