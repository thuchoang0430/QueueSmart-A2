# QueueSmart A2 Front-End Project

This repository contains the front-end UI for QueueSmart, a smart queue management application.

## Technology Used
- HTML
- CSS
- JavaScript

## Project Scope
This assignment is front-end only. Backend functionality is not required, so mock/static data will be used.

## Required Screens
- Login / Registration
- User Dashboard
- Join Queue
- Queue Status
- History
- Admin Dashboard
- Service Management
- Queue Management

## Team Workflow
Each team member will work on their assigned screens and commit their own changes to GitHub.

## How to Run
Open `index.html` in a web browser.
